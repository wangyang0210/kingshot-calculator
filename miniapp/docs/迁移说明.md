# KingshotData 微信小程序（uni-app + Vue3）迁移说明

## 项目结构
- 根目录：miniapp
- 关键文件：pages.json、manifest.json、uni.scss、App.vue、main.js
- 页面：pages（主包）与 subpackages（分包）
- 适配层：utils/i18n.js、utils/platform.js

## 页面映射
- 首页：pages/home/index
- 建筑：pages/buildings/index、pages/buildings/detail
- 英雄：pages/heroes/index、pages/hero/detail
- 计算器：subpackages/calculators/pages/*
- 数据库：subpackages/database/pages/index
- 战争学院：subpackages/waracademy/pages/*

## 多语言
- 复用现有 i18n JSON，支持 zh-CN/zh-TW/en/ko/ja
- i18n 接口：setLang/getLang/t

## 导航与参数
- 跳转：uni.navigateTo
- 返回：uni.navigateBack
- 详情页参数：onLoad(options)

## 开放能力
- 订阅消息：wx.requestSubscribeMessage 封装为 subscribe
- 剪贴板/系统信息：clipboard/systemInfo

## 构建与运行
- 目标平台：mp-weixin
- 使用微信开发者工具导入运行
